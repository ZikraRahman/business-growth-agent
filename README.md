# AI Multi-Agent Business Assistant

A production-ready AI business assistant powered by multiple specialized AI agents orchestrated using n8n.

## Features

- AI Chat Interface
- Multi-Agent Routing
- Marketing Agent
- Research Agent
- Conversation Memory
- Modern Responsive UI
- Dark Mode
- Real-time API Integration

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- shadcn/ui
- Framer Motion
- n8n
- OpenAI

## Backend

The backend is implemented in n8n.

The frontend communicates with the webhook defined in the environment variables.

## Development

Install

npm install

Run

npm run dev

Build

npm run build
